基于 RICS-V 指令集的 CPU 设计

集创赛全国一等奖

Fmax: 330MHz  |  Bus: AXI4  |  OS: RT-Thread

本项目致力于打造一颗高主频、可运行操作系统的 32 位 RISC-V 处理器，指令集支持 RV32IMF_Zicsr。实现浮点 FPU，支持单精度浮点运算

  • 时序收敛 (Timing Closure): 针对 Xilinx XC7K325T 平台进行深度优化。通过流水线切割和关键路径逻辑优化，将系统最高运行频率提升至 330MHz。
  • SoC 总线互联: 实现了复杂的 AXI4 总线协议以支持高带宽数据传输，替代传统简单总线，增强了外设拓展能力并成功运行 RT-Thread 操作系统。
  • 架构完整性: 支持单精度浮点运算 (FPU) 及精确中断控制。集成 UART/Timer 等外设，通过 CoreMark 跑分 (16s/10k) 验证了综合性能。

  • 逻辑综合与约束 (Synthesis): 使用 Design Compiler 进行逻辑综合，编写 SDC 文件定义时钟频率、IO 延时及设计规则约束 (DRC)，优化面积与功耗。
  • 物理实现 (P&R): 在 IC Compiler II 中完成 Floorplan 规划、Power Network 构建及 Standard Cell 布局。通过 CTS (时钟树综合) 优化 Clock Skew。

智慧农业视角下的稻谷产量预测

统计建模大赛
R² = 0.9901 Stacking Ensemble Python/SPSS

针对传统单一模型预测精度低的问题，基于国家统计局 25 年的历史数据，构建了基于 Stacking 集成学习策略 的高精度产量预测模型。

  • 模型架构创新: 选取随机森林、AdaBoost、梯度提升树 (GBDT) 等四种差异化模型作为基学习器，有效降低了过拟合风险。最终模型 MSE 降至 0.0012，拟合优度达到 0.9901。
  • 数据驱动归因: 利用前馈神经网络 (Feedforward NN) 进行特征重要性分析，量化了“农业机械总动力”与“智慧农业信息化水平”对产量的具体贡献权重，为农业决策提供数据支持。
